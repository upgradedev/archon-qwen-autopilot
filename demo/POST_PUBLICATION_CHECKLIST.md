# Post-publication checklist — operator only

**This file is operational guidance, not public post copy. Never paste any part of it
into LinkedIn, X, Devpost Community, a blog, or the Devpost description.** Public post
body copy lives only inside the fenced blocks in [`POST_DRAFTS.md`](POST_DRAFTS.md);
that file's final section separately supplies destination image alt text.

## Before copying a draft

- [ ] Complete the exact-release proof in [`BUILD_RECORDING.md`](BUILD_RECORDING.md).
- [ ] Confirm the final submission commit's immutable Node, real-pgvector, browser,
  adversarial, coverage, secret-scan, dependency-audit, CodeQL, and image/SBOM gates.
- [ ] Publish the final video on an unrestricted Public page and replace
  `[PUBLIC_VIDEO_URL]` in the selected draft locally at publication time.
- [ ] Do not reuse historical test totals or a CI URL from an earlier commit.
- [ ] Keep the reviewer token, private contact data, cloud identifiers, unpublished
  account URLs, terminal history, and administrative screenshots out of the post.
- [ ] Select one destination-specific fenced block from `POST_DRAFTS.md`; copy only
  its contents, never this checklist or the Markdown fence.

## Media and claim review

- [ ] Attach only sanitized, rights-cleared current-release media from
  `demo/final-media/`; use the supplied alt text where the platform supports it.
- [ ] Preserve the exact scope: offline deterministic policy eval, two configurable
  real transports, simulated payment/review, and pattern-based advisory scanning.
- [ ] Do not call deterministic eval live-model accuracy, SMTP acceptance recipient
  delivery, JSONL an ERP integration, or structural least agency universal injection
  detection.
- [ ] Preview line breaks, links, images, and alt text on mobile and desktop.

## After publication

- [ ] Open the canonical public URL signed out/incognito and verify every link and
  image without cached credentials.
- [ ] Save the public URL directly in the Devpost draft's optional blog/social field;
  do not commit unpublished account data or secrets to Git.
