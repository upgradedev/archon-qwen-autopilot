Every business drowns in incoming invoices. Each one has to be recorded, validated, checked for duplicates and for amounts that look wrong, and then decided on. It is slow, it is error-prone, and one rule can never be broken: money must never leave the account without a human saying yes.

Archon Autopilot is a human-gated accounts-payable agent. It runs on Qwen — qwen-plus function-calling — live on Alibaba Cloud, and it is grounded in a persistent vendor memory carried over from our Track One MemoryAgent. It takes a messy invoice all the way to a proposed action, then stops and waits for a person.

Here is a real invoice, sent live over HTTPS to the deployed box. Watch the multi-step loop think. Step one: it recalls the vendor's history from pgvector memory. Step two: it validates the invoice against six cross-checks. Step three: it checks for a duplicate. Step four: it computes the variance against past amounts. These are autonomous, side-effect-free steps — the agent gathering evidence, one tool at a time. Only then does it commit to a single terminal action: draft a journal entry, with its reasoning and a confidence of ninety-five percent — and then it deliberately stops.

Nothing has executed. The proposal lands in the approval queue as pending. A human sees the vendor, the amount, the proposed action, and the full trace — how the agent decided, step by step. From here a person can approve it, amend the arguments and then approve, or reject it outright. And when the same invoice arrives twice, the agent recalls the earlier one, confirms the duplicate, and flags it for review instead of paying — exactly the mistake this gate exists to stop.

The same capability is exposed as a Model Context Protocol server with seven tools, plus a formalized catalog of nine custom skills — five autonomous, four human-gated. Any MCP client can drive the very same human-gated workflow.

It is live on Alibaba Cloud, over HTTPS, on real Qwen models, with the code open source under MIT. Real reasoning, real memory, and a human always in the loop.
